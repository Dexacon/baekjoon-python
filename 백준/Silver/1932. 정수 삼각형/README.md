<pre>        7

<p>위 그림은 크기가 5인 정수 삼각형의 한 모습이다.</p>

<p>맨 위층 7부터 시작해서 아래에 있는 수 중 하나를 선택하여 아래층으로 내려올 때, 이제까지 선택된 수의 합이 최대가 되는 경로를 구하는 프로그램을 작성하라. 아래층에 있는 수는 현재 층에서 선택된 수의 대각선 왼쪽 또는 대각선 오른쪽에 있는 것 중에서만 선택할 수 있다.</p>

<p>삼각형의 크기는 1 이상 500 이하이다. 삼각형을 이루고 있는 각 수는 모두 정수이며, 범위는 0 이상 9999 이하이다.</p>

### 입력 

 <p>첫째 줄에 삼각형의 크기 n(1 ≤ n ≤ 500)이 주어지고, 둘째 줄부터 n+1번째 줄까지 정수 삼각형이 주어진다.</p>

### 출력 

 <p>첫째 줄에 합이 최대가 되는 경로에 있는 수의 합을 출력한다.</p>


  ### 🧠 배운 점 & 설계 과정
1. **문제 해석**: 
   - 첫째 줄에 삼각형의 크기 N이 주어지고, 둘째 줄부터 n+1번째 줄까지 정수 삼각형이 주어진다.
   - 삼각형은 숫자로된 피라미드 형태이며 맨 위층부터 아래층에 있는 수 중 하나를 선택하면서 아래층으로 내려온다.
   - 선택된 수의 합이 최대가 되는 경로를 구하여라.
   - 아래층으로 내려올때는 대각선 왼쪽 또는 대각선 오른쪽에 있는 수만 선택 가능하다.
2. **설계**:
   - 우리는 피라미드를 리스트 형태로 만들었기 때문에 인덱스로 보면 j가 0인 숫자는 자신 바로 위의 숫자만, j가 i인 숫자는 왼쪽 대각선 위의 숫자만을 선택 가능하다.
   - 양쪽 끝이 아닌 사이에 있는 숫자들은 선택할수 있는경우가 자기 바로 위 숫자와 왼쪽 대각선 위의 숫자 2가지이다.
   - 위에서 부터 아래쪽으로 값을 더하면서 내려오는데 위의 조건을 지키면서 내려온다.(중간에 있는 값들은 선택지가 2개이므로 2개중 더 큰값을 가지는 숫자를 선택해서 내려온다.)
   - N-1번째까지 반복하고 max(triangle[N-1])를 통해 선택된 수의 합이 최대가 되는 값을 구한다.
3. **실수 & 해결**:
   - 중간에 있는 숫자는 바로 위 숫자와 왼쪽 대각선의 숫자 2가지 경우가 있었는데 경우를 모두 구하려고 했다.(경우의수가 2^n으로 늘어난다.)
   - 둘중 더 낮은 값은 버리고 더 높은 값을 선택해서 가지고 와서 해결했다.(다이나믹 프로그래밍)

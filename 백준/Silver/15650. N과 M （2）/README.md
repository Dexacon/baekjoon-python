### 문제 설명

<p>자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.</p>

<ul>
	<li>1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열</li>
	<li>고른 수열은 오름차순이어야 한다.</li>
</ul>

### 입력 

 <p>첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)</p>

### 출력 

 <p>한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.</p>

<p>수열은 사전 순으로 증가하는 순서로 출력해야 한다.</p>

 ### 🧠 배운 점 & 설계 과정
1. **문제 해석**: 
   - 자연수 N과 M이 주어졌을때 1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열중 오름차순으로 정렬된 수열을 구하라.
   - N과 M(1) 문제에서 조건을 추가하여야 할것 같다.
2. **설계**:
   - N과 M(1) 기본 구조에서 오름차순인 수열만 골라서 result에 추가한다.
   - dfs 함수에서 start를 추가한다.
   - 다음 숫자를 고를때 dfs(start+1,depth+1)을 호출하여 이 전에 고른 숫자보다 반드시 큰 수부터 고르도록 한다.
   - 4 4 같은 경우에는 1 2 3 4 가 아닌 경우에 depth가 M에 도달하지 못해서 한 수열만 출력된다.
   - 이 전에 고른 수보다 반드시 큰 수를 고르기 때문에 사용한 숫자인지 아닌지 체크 할 필요가 없다.
3. **실수 & 해결**:
   - 처음에는 인수 start를 추가해서 할 생각을 못하고 조건문을 통해 이 전에 고른 수보다 커야만 고를 수 있도록 설정했다.

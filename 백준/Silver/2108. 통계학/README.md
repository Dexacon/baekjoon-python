### 문제 설명

<p>수를 처리하는 것은 통계학에서 상당히 중요한 일이다. 통계학에서 N개의 수를 대표하는 기본 통계값에는 다음과 같은 것들이 있다. 단, N은 홀수라고 가정하자.</p>

<ol>
	<li>산술평균 : N개의 수들의 합을 N으로 나눈 값</li>
	<li>중앙값 : N개의 수들을 증가하는 순서로 나열했을 경우 그 중앙에 위치하는 값</li>
	<li>최빈값 : N개의 수들 중 가장 많이 나타나는 값</li>
	<li>범위 : N개의 수들 중 최댓값과 최솟값의 차이</li>
</ol>

<p>N개의 수가 주어졌을 때, 네 가지 기본 통계값을 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 수의 개수 N(1 ≤ N ≤ 500,000)이 주어진다. 단, N은 홀수이다. 그 다음 N개의 줄에는 정수들이 주어진다. 입력되는 정수의 절댓값은 4,000을 넘지 않는다.</p>

### 출력 

 <p>첫째 줄에는 산술평균을 출력한다. 소수점 이하 첫째 자리에서 반올림한 값을 출력한다.</p>

<p>둘째 줄에는 중앙값을 출력한다.</p>

<p>셋째 줄에는 최빈값을 출력한다. 여러 개 있을 때에는 최빈값 중 두 번째로 작은 값을 출력한다.</p>

<p>넷째 줄에는 범위를 출력한다.</p>

 ### 🧠 배운 점 & 설계 과정
1. **문제 해석**: 
   - 첫째 줄에 수의 개수 N이 주어지고 N은 홀수이다.
   - 그 다음 줄부터 N개의 정수들이 주어진다.
   - 첫째 줄부터 넷째 줄까지 각 줄마다 산술평균, 중앙값, 최빈값, 범위를 출력한다.
   - 산술평균과 중앙값 범위는 일반적인 리스트로 풀 수 있을것같으나 최빈값은 딕셔너리를 써야 할 것 같다.
2. **설계**:
   - 최빈값을 제외한 모든값은 리스트를 이용해 풀 수 있으나 최빈값은 해당 값이 입력된 횟수를 저장해야 하므로 딕셔너리를 사용한다.
   - 입력받은 모든 값을 리스트와 딕셔너리에 넣고 정렬한다.(리스트는 오름차순, 딕셔너리는 우선순위를 1. 값이 입력된 횟수 내림차순, 2. 값의 오름차순 으로 정렬한다.)
   - 산술평균은 round를 이용하여 소수점 첫째자리에서 반올림하고 중앙값은 N//2로 인덱스를 구한다. 범위는 nums을 정렬했으므로 nums[-1]과 nums[0]의 차로 구한다.
   - 최빈값은 딕셔너리에 들어있는 값이 2개 이상이고 최빈값이 여러개일때는 두번째로 작은값을 출력하고 나머지 경우는 최빈값을 출력한다.
3. **실수 & 해결**:
   - 딕셔너리만 쓰려고 했다가 중앙값같은 경우에는 다른값이 나오게됨
   - 리스트와 딕셔너리를 둘 다 사용하므로써 해결 [Silver II] 통계학 - 2108

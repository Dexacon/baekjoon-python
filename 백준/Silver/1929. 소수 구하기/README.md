### 문제 설명

<p>M이상 N이하의 소수를 모두 출력하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 자연수 M과 N이 빈 칸을 사이에 두고 주어진다. (1 ≤ M ≤ N ≤ 1,000,000) M이상 N이하의 소수가 하나 이상 있는 입력만 주어진다.</p>

### 출력 

 <p>한 줄에 하나씩, 증가하는 순서대로 소수를 출력한다.</p>

### 🧠 배운 점 & 설계 과정
1. **문제 해석**: 
   - 두 숫자 사이의 소수를 모두 출력
   - 정수론의 소수판정과 에라토스테네스의 체를 사용
2. **설계**:
   - 숫자 N까지의 인덱스를 가진 Boolean 리스트를 생성하고, 모든 값을 True(소수라고 가정)로 초기화
   - 2부터 n-1까지 검사하는 대신 n의 제곱근+1까지만 검사하여 시간복잡도를 단축
   - 에라토스테네스의 체 알고리즘을 통해 소수가 아닌 인덱스의 값을 False로 변경
   - M부터 N까지의 범위를 순회하며, 리스트의 값이 True로 남아있는 인덱스(소수)들만 골라 출력
3. **실수 & 해결**:
   - 첫 시도는 소수판정 함수를 두 숫자 사이의 수에 모두 적용하여 출력 -> 메모리는 적게 쓰지만 실행속도가 매우 느림
   - 직전 문제에서 학습한 에라토스테네스의 체를 사용하면 실행속도가 매우 빨라진다는 걸 생각함
   - 나머지 연산을 통해 리스트 전체를 확인하려 했으나 속도가 느려짐
   - 따라서 range(i*i, N+1, i) 를 활용해 소수의 배수만 선택적으로 False 지정하여 실행속도를 개선

### 문제 설명

<ul>
	<li>골드바흐의 추측: 2보다 큰 짝수는 두 소수의 합으로 나타낼 수 있다.</li>

<p>짝수 N을 두 소수의 합으로 나타내는 표현을 골드바흐 파티션이라고 한다. 짝수 N이 주어졌을 때, 골드바흐 파티션의 개수를 구해보자. 두 소수의 순서만 다른 것은 같은 파티션이다.</p>

### 입력 

 <p>첫째 줄에 테스트 케이스의 개수 T (1 ≤ T ≤ 100)가 주어진다. 각 테스트 케이스는 한 줄로 이루어져 있고, 정수 N은 짝수이고, 2 < N ≤ 1,000,000을 만족한다.</p>

### 출력 

 <p>각각의 테스트 케이스마다 골드바흐 파티션의 수를 출력한다.</p>

 ### 🧠 배운 점 & 설계 과정
1. **문제 해석**: 
   - 짝수 N을 두 소수의 합으로 나타내는 표현을 골드바흐 파티션이라고 함
   - 짝수 N의 골드바흐 파티션의 개수 구하기
   - 에라토스테네스의 체 활용 
2. **설계**:
   - 에라토스테네스의 체를 한 번만 생성하기 위해 N의 최댓값인 1,000,000의 에라토스테네스의 체 생성
   - 중복 방지와 합의 대칭으로 N/2까지만 조사
   - 2부터 N/2까지 반복문(i)을 수행하면서 N-i 인덱스가 참인 확인(1은 소수가 아니기 때문에 제외)
3. **실수 & 해결**:
   - 에라토스테네스의 체를 만들어놓고 N-i를 따로 소수판정 함수를 만들어 검사하려고 함
   - 마지막 반복문에서 연산이 너무 중첩되어 시간초과 발생
   - N-i 인덱스가 참인지 에라토스테네스의 체로 확인함으로써 해결

### 문제 설명

<p>0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.</p>

### 출력 

 <p>첫째 줄에 N!을 출력한다.</p>

 ### 🧠 배운 점 & 설계 과정
1. **문제 해석**: 
   - 팩토리얼을 python으로 구현한다.
   - n의 팩토리얼은 n부터 n-1, n-2,n-2 --- 1 까지 모두 곱한 값이다.
2. **설계**:
   - result를 1로 설정하고 while n > 0으로 result에 n의 값을 곱하고 n-=1을 반복문이 깨질때까지 진행한다.
